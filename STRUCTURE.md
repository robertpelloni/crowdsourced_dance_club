# Project Structure & Submodules

## Directory Layout
- `src/`: Main source code for the Python AI Conductor Server.
  - `main.py`: FastAPI server, WebSocket management, and vibe-check logic.
  - `init_db.py`: Database initialization script.
- `external/`: External dependencies integrated as submodules.
  - `auto_dj_script/`: High-fidelity offline audio mixing engine.
- `tests/`: Unit and integration tests.
- `tracks.db`: SQLite database for track metadata.
- `requirements.txt`: Python dependencies.
- `VERSION.md`: Centralized version number.

## Submodules

### Auto DJ Script
- **URL:** https://github.com/robertpelloni/auto_dj_script
- **Location:** `external/auto_dj_script`
- **Description:** A batch audio processor that uses simulated annealing and advanced DSP to create studio-grade DJ sets.
- **Role in Project:** Acts as the offline "heavy lifter" for rendering high-quality master files from the crowdsourced playlists generated by the Conductor Server.
- **Key Technologies:** Librosa, NumPy, SciPy, FFmpeg, Phase Vocoder, Simulated Annealing.
- **Version Tracking:** Tracked via git submodule commit hash.

## Libraries & Dependencies
- **FastAPI:** High-performance web framework for the Conductor Server.
- **Uvicorn:** ASGI server for running FastAPI.
- **Pydantic:** Data validation using Python type hints.
- **Websockets:** Real-time communication with mobile clients.
- **SQLite3:** Lightweight persistent storage for track metadata.
- **Pytest:** Testing framework for ensuring logic correctness.
